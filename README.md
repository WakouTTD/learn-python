# python3入門

## local環境設定

### Anaconda

- Anacondダウンロード

[Anaconda ダウンロードページ](https://www.anaconda.com/distribution/#download-section)

- ダウンロードしたパッケージをクリックして手順にしたがってAnacondインストール

- Anacondaによって、python3がインストールされたことの確認

ターミナルで`python3`とコマンドを打って対話型シェルを呼び出す
